# Interactive Form Validation

## Overview
This project demonstrates a functional interactive form built using HTML, CSS, and JavaScript. The form performs client-side validation before submission.

## Features
- User-friendly registration form
- Client-side validation
- Email format checking
- Password length validation
- Dynamic success and error messages
- Responsive and clean UI

## Files Included

```text
Interactive-Form-Validation/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Validation Rules

### Name
- Cannot be empty

### Email
- Must contain '@'

### Password
- Minimum 6 characters
